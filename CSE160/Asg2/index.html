Features:

  • RGB preview: next to RGB sliders is the a preview of the color selected.
  • Clear: clears any drawn shapes.
  • Point, Triangle, Circle: Brush shapes.
  • Cont: when checked, continuously draws shapes (without mouse click).
  • Butterfly: shows the drawing (turn opacity to 100). Note that these shapes were algorithmically derived from the the underlying butterfly image. Read in img/butterfly.js on the generation details.
  • Size: the size of the brush.
  • # Segments: the number of segments used for Circle brush.
  • Opacity: controls the opacity of the drawing layer. Turn to 100 to hide the underlying image.
  • Image src: enter a valid url and image will show up in the background. If the image is invalid it will default to the butterfly image. Copy and paste the following as an example: https://www.w3schools.com/css/img_forest.jpg
